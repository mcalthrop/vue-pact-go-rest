# Instructions

Following are instructions for building the required app:

- read the requirements in [`REQUIREMENTS.md`](./REQUIREMENTS.md)
- devise a plan that contains a list of tasks and subtasks that need to be performed
- each subtask should be small enough to be implemented in a single PR that is not excessively large
- the plan will be stored in [`PLAN.md`](./PLAN.md)
- as tasks and subtasks are completed, update `PLAN.md` to mark completed tasks
